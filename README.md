# Дипломный проект: Парсер PLX-файлов + Django сайт

## Описание
Проект для парсинга учебных планов в формате PLX и их отображения через Django-сайт.

## Структура
- `src/` — исходный код парсера и main.py
- `tests/` — тесты для парсера
- `education_site/` — Django-проект

## Запуск
```bash
python src/main.py
